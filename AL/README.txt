Use run_experiment_eurlex.sh script to run Active Learning. 
Provide path to dataset folder containing train.csv, dev.csv and test.csv as well as Active Learning Strategy.
Choose Active Learning strategy from: "random", "alps", "dal", "cvirs" and "subword"

Please refer to http://www.lrec-conf.org/proceedings/lrec2022/pdf/2022.lrec-1.490.pdf for details about Active Learning.
